# `csv2kirby`
> Generate Kirby content from a CSV

<br>

## Installation

```
$ npm install --global arnaudjuracek/csv2kirby
```

## Usage

```console
$ csv2kirby <INPUT.csv> <CONTENT-DIRECTORY>
```

## Development

```console
$ git clone https://github.com/arnaudjuracek/csv2kirby
$ cd csv2kirby && npm install
```
